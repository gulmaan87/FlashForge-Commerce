
**FlashForge Commerce** is a production-grade flash-sale checkout and reliability platform designed to simulate high-concurrency e-commerce traffic, prevent overselling, handle idempotent payment callbacks, and demonstrate autoscaling, observability, and resilience under burst demand.

## Core Goals
- Prevent overselling using TTL-based inventory reservation
- Support async checkout orchestration
- Handle duplicate payment callbacks safely
- Demonstrate Kubernetes autoscaling
- Expose production-style metrics, logs, and load testing results
